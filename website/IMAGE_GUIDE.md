# QueueVision Website Image Guide

This website is fully functional without custom images, but adding these assets will significantly improve the premium winter-tech look.

## 1) Hero Background Visual
- Path suggestion: `website/assets/hero-alpine-tech.png`
- Use in: Hero section background or right-side visual card backdrop.
- Content: A high-altitude ski resort panorama with subtle digital/technical overlay feel.
- Style: Clean, bright winter tone (icy blue and white), not dark cinematic.
- Recommended size: 2400 x 1400 px.

## 2) Technical Camera-at-Lift Photo
- Path suggestion: `website/assets/lift-camera-node-a.png`
- Additional angle: `website/assets/lift-camera-node-b.png`
- Use in: Technology section (Layer 1: Lift Edge Sensors).
- Content: Lift entry area with visible mounted camera hardware.
- Purpose: Reinforce real-world system installation and credibility.
- Recommended size: 1600 x 1000 px.

## 3) AI Operations Dashboard Mockup
- Path suggestion: `website/assets/queuevision-dashboard.png`
- Use in: Economics or Technology section as a product proof visual.
- Content: Resort map, queue times per lift, trend chart, operations panel.
- Style: High-level B2B SaaS UI, technical and minimal.
- Recommended size: 1800 x 1100 px.

## 4) Mobile App Map Screen
- Path suggestion: `website/assets/mobile-map-ui.jpg`
- Use in: Product positioning or guest experience area.
- Content: Mobile screen with ski map and waiting time labels.
- Purpose: Show customer-facing product value quickly.
- Recommended size: 1080 x 1920 px (portrait) or framed device mockup.

## 5) Reputation / Experience Lifestyle Shot
- Path suggestion: `website/assets/happy-skiers.png`
- Use in: About or trust section.
- Content: Skiers enjoying runs, line-free atmosphere.
- Purpose: Connect operational optimization to emotional customer benefit.
- Recommended size: 1800 x 1200 px.

## 6) Optional Ambient Pattern
- Path suggestion: `website/assets/snow-topo-pattern.png`
- Use in: Global background texture overlay.
- Content: Very subtle snow/topographic pattern at low opacity.
- Recommended size: tileable PNG, around 600 x 600 px.

## Photo and Licensing Guidance
- Use only licensed stock or your own original photography.
- Keep color grading consistent: cool daylight, sharp contrast, clean whites.
- Avoid crowded images that hide lifts or queue context.
